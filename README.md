# EtsAl

ETS abstraction layer and assets keeper.

## Installation

```elixir
def deps do
  [
    {:ets_al, "~> 0.1.1"}
  ]
end
```

## Usage

Start the EtsAl.Keeper under your supervision tree in the right place to keep your ETS assets.

```elixir
require EtsAl.Keeper
```
